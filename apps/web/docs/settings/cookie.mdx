# cookie

**cookie** is an object with the following properties:

<br/>

- **key** - the name of the cookie
- **daysToExpire** - the number of days until the cookie expires

<br/>

## Default
```tsx
key: 'exit-intent'
daysToExpire: 30
```

<br />

## Example
```tsx
useExitIntent({
  cookie: {
    key: 'exit-intent',
    daysToExpire: 30
  }
})
```
